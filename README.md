# service
### Diesel+SQL
  
# How to run diesel
  * Build server repo
  ```
   cargo build 
  ```
  * Insert data to Database
  ```
   cargo run insert 
  ```
  * Delete data to Database
  ```
   cargo run delete <id> 
  ```
 * Show specific data from Database
  ```
   cargo run query <id> 
 ```
 * Update data in Database
  ```
   cargo run update_pass <id> <pass> 
  ```
